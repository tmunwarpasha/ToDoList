# ToDoList
To create ToDo Lists and tracking them
This app helps plan your calendar by creating and scheduling tasks
you can track your tasks by date
All tasks will be displayed in a single page and this is a single page application
You can plan both personal and official tasks, it will save the data to different file depending upon the selection
Pending tasks will be shown on the left
Today's tasks will be shown in the middle
Future tasks will be shown on the right side

Tools/Technology used to develop this app is

1. JSF 2.x 
2. Java 8
3. Can be deployed on Tomcat 9 or any webcontainer
